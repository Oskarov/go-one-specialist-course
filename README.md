### Description

* LEC1 - hello world
* LEC2 - инициализация, присваивание, вывод переменных - fmt.print
* LEC3 - ввод и вывод значений - fmt.scan
* LEC4 - типы данных - unsafe.SizeOf
* LEC5 - условный оператор
* LEC6 - циклы
* LEC7 - switch
* LEC8 - массивы
* LEC9 - срезы - slice
